# FeeZero 관리자 가이드

## 관리자 페이지 접속

### 로컬 환경
```
http://localhost:3000/admin
```

### 샌드박스 환경
```
https://3000-iaz6hxe6dnh7awzc4b7fz-5634da27.sandbox.novita.ai/admin
```

**중요**: 관리자 페이지는 아이디/비밀번호 없이 URL만으로 접속 가능합니다.
메인 사이트에는 관리자 접속 버튼이 없으며, URL을 직접 입력해야 합니다.

## 관리자 기능

### 1. 회원 관리 탭

#### 기능
- 전체 회원 목록 조회
- 회원 등급별 필터링 (무료/유료/관리자)
- 회원 추가
- 회원 정보 편집
- 회원 삭제

#### 회원 추가 방법
1. "회원 추가" 버튼 클릭
2. 필수 정보 입력:
   - 이메일 (필수)
   - 닉네임 (필수)
   - 전화번호 (선택)
   - 회원 등급 선택: 무료 회원 / 유료 회원 / 관리자
3. "저장" 버튼 클릭

#### 회원 편집 방법
1. 회원 목록에서 "편집" 버튼 클릭
2. 정보 수정
3. "저장" 버튼 클릭

#### 회원 삭제 방법
1. 회원 목록에서 "삭제" 버튼 클릭
2. 확인 대화상자에서 "확인" 클릭

### 2. 공지사항 관리 탭

#### 기능
- 전체 공지사항 목록 조회
- 공지사항 추가
- 공지사항 편집
- 공지사항 삭제
- 이미지 업로드 지원

#### 공지사항 추가 방법
1. "공지 추가" 버튼 클릭
2. 정보 입력:
   - 제목 (필수)
   - 내용 (필수)
   - 우선순위 선택: 일반 / 중요 / 긴급
   - 이미지 업로드 (선택)
3. "저장" 버튼 클릭

#### 이미지 업로드
- PC에서 이미지 파일 선택 가능
- 지원 형식: JPG, PNG, GIF
- 공지사항과 함께 표시됨

#### 공지사항 편집 및 삭제
- 편집: 공지사항 카드에서 연필 아이콘 클릭
- 삭제: 공지사항 카드에서 휴지통 아이콘 클릭

### 3. 게시물 관리 탭

#### 기능
- 전체 게시물 목록 조회
- 게시물 추가
- 게시물 편집
- 게시물 삭제
- 게시물 유형 및 상태 관리

#### 게시물 유형
- **기사 (article)**: 일반 기사
- **뉴스 (news)**: 뉴스 및 업데이트
- **블로그 (blog)**: 블로그 포스트
- **가이드 (guide)**: 사용자 가이드
- **FAQ**: 자주 묻는 질문

#### 게시물 상태
- **초안 (draft)**: 작성 중
- **게시됨 (published)**: 공개됨
- **보관됨 (archived)**: 보관됨

#### 게시물 추가 방법
1. "게시물 추가" 버튼 클릭
2. 정보 입력:
   - 제목 (필수)
   - 내용 (필수)
   - 게시물 유형 선택
   - 상태 선택
3. "저장" 버튼 클릭

### 4. 통계 대시보드 탭

#### 표시 항목
- **전체 회원**: 등록된 전체 회원 수
- **유료 회원**: Premium 회원 수
- **전체 프로젝트**: 등록된 프로젝트 수
- **공지사항**: 등록된 공지사항 수

통계는 실시간으로 업데이트됩니다.

## 회원 등급 시스템

### 무료 회원 (Free)
- 메뉴만 볼 수 있음
- 기능 사용 불가
- 프로젝트 등록 불가
- 입찰 불가
- 메시지 발송 불가

### 유료 회원 (Premium)
- 모든 기능 사용 가능
- 프로젝트 등록 가능
- 입찰 가능
- 메시지 발송 가능
- 포트폴리오 등록 가능

### 관리자 (Admin)
- 모든 기능 + 관리자 패널 접근
- 회원 관리
- 공지사항 관리
- 게시물 관리
- 통계 조회

## 데이터베이스 테이블

### admin_logs
관리자의 모든 작업을 기록합니다.
- 회원 생성/편집/삭제
- 공지사항 생성/편집/삭제
- 게시물 생성/편집/삭제

### site_posts
사이트 게시물을 관리합니다.
- 제목, 내용
- 게시물 유형
- 상태
- 작성자 정보

### site_post_translations
게시물의 다국어 번역을 관리합니다.
- 8개 언어 지원
- 제목, 내용, 요약 번역

## API 엔드포인트

### 회원 관리
- `GET /admin/api/users` - 전체 회원 목록
- `POST /admin/api/users` - 회원 추가
- `PUT /admin/api/users/:id` - 회원 수정
- `DELETE /admin/api/users/:id` - 회원 삭제

### 공지사항 관리
- `GET /admin/api/announcements` - 전체 공지사항 목록
- `POST /admin/api/announcements` - 공지사항 추가
- `DELETE /admin/api/announcements/:id` - 공지사항 삭제

### 게시물 관리
- `GET /admin/api/posts` - 전체 게시물 목록
- `POST /admin/api/posts` - 게시물 추가
- `DELETE /admin/api/posts/:id` - 게시물 삭제

### 통계
- `GET /admin/api/stats` - 통계 데이터

## 문제 해결

### 관리자 페이지가 보이지 않을 때
1. URL 끝에 슬래시(/)가 없는지 확인: `/admin` (O), `/admin/` (X)
2. 서버가 실행 중인지 확인: `pm2 list`
3. 빌드가 완료되었는지 확인: `npm run build`

### 데이터가 로드되지 않을 때
1. 브라우저 콘솔에서 에러 확인
2. API 엔드포인트 접근 가능 여부 확인
3. 데이터베이스 마이그레이션 적용 여부 확인

### 이미지 업로드가 작동하지 않을 때
현재 이미지 업로드는 프론트엔드만 구현되어 있습니다.
백엔드 구현이 필요합니다 (Cloudflare R2 또는 다른 스토리지 서비스).
